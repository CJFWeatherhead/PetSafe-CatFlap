# Code of Conduct

## Our Standards

This is a small, focused project for documenting and improving an alternative firmware for PetSafe cat flaps. We expect all participants to:

- Be respectful and professional in communications
- Focus on technical discussions and improvements
- Report issues and bugs constructively
- Share testing results honestly
- Respect that this is not an actively maintained project

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repository (issues, pull requests, discussions)
- Any communication related to this project

## Reporting

If you experience or witness unacceptable behavior, please report it by opening an issue or contacting the repository owner.

## Attribution

This Code of Conduct is adapted for a minimal, community-driven embedded systems project.
